# 💡 AI 주식 분석 비서 (Stock-Agent)

실시간 **네이버 뉴스 API** 검색 결과와 **Google Gemini (gemini-2.5-flash) 모델**을 결합하여, 주식 종목에 대한 체계적인 5단계 에이전트 분석 및 프리미엄 브리핑 보고서를 제공하는 Streamlit 기반의 웹 애플리케이션입니다.

---

## 🚀 주요 특징 (5단계 에이전트 워크플로우)

본 애플리케이션은 단일 질문 처리 방식에서 벗어나, 아래와 같은 **5단계 파이프라인**을 통해 정교한 투자 보조 분석을 수행합니다.

1. **1단계: 뉴스 수집 (News Collection)**
   - 네이버 뉴스 검색 API를 통해 입력된 종목에 대해 정확도순(`sim`) 15건, 최신순(`date`) 15건을 교차 수집하여 최신성과 연관성을 모두 확보합니다.
2. **2단계: 뉴스 필터링 (News Filtering)**
   - **저품질 뉴스 제외**: 제목 8자 미만, 본문 요약 20자 미만 기사는 분석 제외합니다.
   - **오래된 뉴스 제외**: 수집된 뉴스 중 가장 최신 기사의 날짜 기준 **14일(2주)이 경과한 뉴스**를 제외하여 최신 트렌드만 유지합니다.
   - **🔑 중복 뉴스 제거 (Dedupe Key)**: `도메인 + 경로 + 공백제거 소문자 제목의 앞 40글자`로 고유 식별 키를 생성하여 여러 언론사에 복사되어 퍼진 중복 뉴스를 완벽히 제거합니다.
3. **3단계: 감성 분석 (Sentiment Analysis)**
   - 필터링된 개별 기사를 대상으로 Gemini를 호출하여 **"호재", "악재", "중립"** 중 하나로 판정하고, 이에 대한 **상세한 분석 근거(이유)**를 도출합니다.
4. **4단계: 리스크 분석 (Risk Analysis)**
   - 수집된 기사 내용 및 감성 결과를 종합 분석하여 투자자가 주의 깊게 보아야 할 **거시경제/실적/경쟁/변동성 관점의 리스크 요인 3~5가지**를 추출합니다.
5. **5단계: 최종 브리핑 생성 (Final Briefing)**
   - 요약 정보, 호재/악재 분석, 리스크 관점을 취합하여 구조화된 종합 보고서를 작성합니다.
   - **금융 심의 필터**: "매수, 매도, 강력 추천" 등 편향된 직접 투자 유도 표현을 중립적 단어(예: "긍정적 전망", "리스크 요인")로 순화하고, 최하단에 **투자 책임 면책 조항(DISCLAIMER)**을 고정 부착합니다.

## 🎨 화면 UI 구성

네이버 금융 테마(Green) 스타일이 가미된 세련된 인터페이스를 제공합니다.

* **Agent Flow (실시간 상태바)**: 분석을 요청하면 1단계부터 5단계까지의 상태 정보가 순차적으로 녹색 체크로 업데이트되어 실시간 진행률을 알 수 있습니다.
* **감성 통계 배너**: 최종 보고서 상단에 **호재 🟢 / 악재 🔴 / 중립 🟡 건수** 메트릭 배너가 크게 표시됩니다.
* **리스크 요약 박스**: ⚠️ 경고 아이콘 스타일 박스 내에 위험 요소를 가독성 높은 목록 형태로 렌더링합니다.
* **상세 뉴스 카드 목록**: 화면 하단에 수집된 기사 카드를 배치하고, **AI가 판단한 개별 기사의 감성 태그** 및 **판정 근거**, **원문 기사 이동 버튼**을 표시합니다.
* **실시간 히스토리 저장**: 분석한 종목은 로컬 파일(`chat_history.json`)에 실시간 누적 백업되며, 사이드바를 통해 이전 분석 기록(상세 뉴스 카드 포함)을 언제든지 정상 복구하여 다시 로드할 수 있습니다.

## 🛠️ 사전 준비 (Prerequisites)

이 프로젝트를 구동하기 위해 API Key 설정이 필요합니다.

### 1. 환경 변수 (`.env`) 설정
프로젝트 루트 디렉토리에 `.env` 파일을 생성하고 아래 키를 입력합니다.
```env
# Google Gemini API Key
GOOGLE_API_KEY=your_gemini_api_key_here

# Naver API Client ID & Secret
NAVER_CLIENT_ID=your_naver_client_id_here
NAVER_CLIENT_SECRET=your_naver_client_secret_here
```
*네이버 API 키는 [네이버 개발자 센터](https://developers.naver.com/)에서 검색(뉴스) API 권한을 신청해 발급받을 수 있습니다.*

### 2. 파이썬 의존 패키지 설치
터미널에서 아래 명령을 실행하여 필수 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

---

## 🏃‍♂️ 실행 방법

터미널에서 아래 명령어를 실행하여 로컬 Streamlit 웹 서버를 구동합니다.
```bash
streamlit run app.py
```
서버 가동 후 브라우저 창에서 `http://localhost:8501`로 접속해 사용하실 수 있습니다.
